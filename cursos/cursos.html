Cursos

Os cursos aqui disponibilizados tem como objetivo auxiliar os alunos que cursam o primeiro semestre de análise e desenvolvimento de sistemas.

Curso de Padlet

De um modo simplificado o padlet é um mural virtual que possibilita a criação de post-its e a organização de tarefas diárias, estudos, compromissos, etc...

Nesse curso, você vai aprender a criar posts, comentar, curtir, configurar e todo o resto que a ferramenta possibilita. Algumas atividades do curso pedirão essa ferramenta como meio de postagem, então é altamente recomendável conhecer todas suas funcionabilidades.

Curso Jamboard

O Google Jamboard nada mais é que uma lousa digital. Uma lousa interativa usada por professores para escrever, desenhar, incluir notas, abrir apresentações e muito mais.

Apesar de ser parecido com o padlet o Google Jamboard tem muito mais funcionalidades. Também é importante se atentar a essa ferramenta pois também existem atividades que os professores exigirão o uso do Google Jamboard.

Curso Bases Numéricas (Binário, Octal, Hexadecimal)

Acha difícil converter de octal para binário? E de hexadecimal para octal? Pois nesse curso de Bases Numéricasvocê vai aprender com facilidade.

Com uma didática excepcional, o professor Gustavo Guanabara torna o assunto desse curto fluído e de fácil entendimento. São quatro aulas que irão agregar muito ao seu conhecimento!

Curso Lógica de Programação

O que é um programador sem esse curso tão básico? A porta de entrada para qualquer linguagem de programação sem dúvida é a lógica de programação.

Com uma didática simples, porém completa, o professor Gustavo Guanabara ensina essa matéria tão importante através do portugol, um software acadêmico.

Curso de Redes de Computadores

Tá achando essa matéria difícil? Não sabe o que significa sistemas distribuídos? Esse curso te responde tudo e muito mais sobre redes de computadores.

Curso PHP para Iniciantes

Você que deseja se tornar um desenvolvedor fullstack ou simplesmente gostaria de aprender sobre back end e sua funcionalidades, esse curso é pra você!

Aqui você vai aprender tudo o que rola por trás daquela página bonitinha que você acessa, envia mensagem, faz pagamentos ou até saca o seu dinheiro.

Curso Git e Github

Sério! Se você é desenvolvedor iniciante ou avançado e ainda não usa git e github, eu tenho uma péssima notícia para você e sua carreia...

Com o git vai ficar muito mais fácil atualizar seus projetos sem medo de perde-los. Com o github você vai poder exibir seu código para o mundo!!
